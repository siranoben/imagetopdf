Smart PDF Converter

A fast and simple web tool that converts images to PDF and PDF to images directly in the browser.

Live Demo

https://img2pdf-hazel.vercel.app/

What it does
Convert images (JPG, PNG, WEBP) to PDF
Convert PDF pages to images
Works fully in the browser (no uploads)
Features
Drag & drop support
Reorder images before converting
Page size options (Fit, A4)
Optional compression
Multi-language support
Instant download
Privacy

No files are uploaded.
Everything runs locally in your browser.

Tech

HTML, CSS, JavaScript, jsPDF, PDF.js
